Logging In/Registering

            When the program is first launched you will be greeted with the screen below. If you are a first-time user, select the register button to create a new account with the system. If you have already registered, log in with the username and password you created during the registration process. This feature allows multiple users on the same computer to keep their medical data separate and private.


 

Once you have selected the register button you will be kindly greeted with the following screen. Then enter your desired username and password combination to create a new account with your data. After selecting the register button once again and submitting your account, log-in with the instructions given at the beginning of this section.


 

Main Screen

            After logging in you will be taken to the main screen. The informative and intuitive design of the main screen is packed full of several beneficial features that will allow you to track and understand your medical information. On the left under the Medical Data tab you will see a list of medical test types that have been pre-loaded into the system. Adding a result is easy,


simply press the add result button located at the bottom of your screen. Use the drop-down menu to select the desired test type and then enter the date and score. When entering the date please note that the format is in dd/mm/yy and is not designed for dates prior to the year 2000.

 


 

Once you have added your very first test result, you can view the test by clicking on its test type and then selecting it under the Test Dates tab. This will display your information in the text box located on the right of the screen as well as displaying the tests trend in the graph just below it. To delete a result, select the test as mentioned above and then press the Delete Result button on the bottom of the screen. Although the program is designed to automatically sort and save your results, if required it can be performed manually using the Sort and Save buttons on the top of the screen. If at any time assistance is needed, click the Help button located at the top of the screen. Medical data can use confusing terminology and units, but the program has a built-in definition guide located at the bottom of the screen to help the user. To open simply click the Open Definitions button and the following information will be displayed.

 


** It is highly recommended that the Close Program button is used to exit the program to avoid any possible corruption of data

 

Warnings and Suggestions Screen

            Using the latest generation technology and state of the art medical research, this screen provides the user with an in-depth analysis of their current and past health condition. To access the screen just select the Launch Warnings Screen button and the following screen will appear. To use the interface, follow the clear on-screen instructions and select the search button to begin. The screen consists of three text boxes, two pertaining to warnings and a third on the right showing possible lifestyle changes in relation to the data. The suggestions are general and are not specific to the user.

**NOTE: This program does not replace a doctors opinion. Always seek direct medical consultation.

 


 

 

Medical Test Type Manager (Advanced Users Only)


            Although the program comes pre-loaded with the most common blood tests, if desired the user can add and track other types of tests using the medical test type manager. To open the manager, press the Medical Test Type Manager button at the bottom left of the screen. The user will be greeted with the following screen. The manager is arranged in a table format with the columns pertaining to the different attributes of each test type. To edit an existing type, double click the attribute and it will turn into a text box. Simply enter the data and hit the enter button to save the data to the table. To add a new type, click the Add Type button at the bottom of the screen and a blank row will be added for the user to edit. To delete a type, select the type in the table and click the Delete Type button. Once completed, to save the new types click the Submit button. To exit without saving, click Cancel.
